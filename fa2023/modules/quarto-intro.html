Quarto workflow

Quarto is a tool for scientific publishing … well, actually … ANY type of publishing where you want to automate the mixing of text, code, data, results, and visuals.

The figure below offers a general idea of a Quarto workflow:

  1. The user creates and edits a QMD file, that is, a text file with the QMD extension.
  2. When ready, the user RENDERs the QMD file into the desired destination file format. Quarto performs all sorts of magic on the way to producing the output file.
code test1.qmd
quarto render test1.qmd

Quarto sample: Gannt chart

gantt
    dateFormat  YYYY-MM-DD
    title       Adding GANTT diagram functionality to mermaid
    excludes    weekends
    %% (`excludes` accepts specific dates in YYYY-MM-DD format, days of the week ("sunday") or "weekends", but not the word "weekdays".)
    section A section
    Completed task            :done,    des1, 2014-01-06,2014-01-08
