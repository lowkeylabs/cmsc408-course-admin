Homework 1 - Quarto and rendering

Quarto is a document compiler, MUCH like compiling a source file to create an executable file. Quarto is a wrapper around Jupyter and Pandoc.

For this assignment, you’ll be modifying an existing qmd file, adding your content, rendering the qmd to html and uploading the HTML to Canvas.

Tip

I definitely recommend the Quarto getting started tutorial! Don’t forget to render the samples I provide in the repository.
